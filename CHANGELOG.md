# Changelog

## Version 2.0.0 (OMEN-Safe Edition) - 2025-11-22

### MAJOR CHANGES - OMEN GAMING HUB PROTECTION

**Previous Version Issues:**
- Broke OMEN Gaming Hub features
- Disabled graphics switching controls
- Affected fan control and temperature monitoring
- Impacted performance management

**New Version Improvements:**
- Only targets telemetry and advertisement services
- OMEN Gaming Hub fully protected and functional
- Fan control works normally
- Temperature monitoring operational
- Graphics switching untouched
- All hardware features remain functional

### What's Blocked (Only 4 Services)

**Telemetry Services:**
- HP Touchpoint Analytics (data collection)
- HP Customer Participation Program (feedback/tracking)

**Advertisement Services:**
- HP Support Solutions Framework (promotional popups)
- HP JumpStart Bridge (promotional launcher)

**Network Blocking:**
- 21+ HP telemetry domains via hosts file
- Firewall rules for HP telemetry programs

### What's Protected (Everything Else)

**Gaming Features:**
- OMEN Gaming Hub (100% functional)
- OMEN Command Center
- Graphics switching and MUX control
- Performance profiles
- Fan control and monitoring
- Temperature sensors

**Hardware Features:**
- Audio (speakers, headphones)
- Touchpad and keyboard
- Display brightness control
- Power management
- WiFi/Bluetooth controls
- All function keys

### Interface Changes

**Simplified Disclaimer:**
- Reduced from 5 pages to 2 pages
- Clear explanation of OMEN protection
- Less intimidating language
- Simple "I ACCEPT" consent

**Simplified Menu:**
- Removed OMEN enable/disable options (not needed)
- Removed general debloat option (too aggressive)
- Removed quick recovery option (not needed)
- Only 2 main options: Block Telemetry & Rollback

### Technical Changes

**Service List:**
- Reduced from 9+ services to only 4 services
- Removed all OMEN-related services from target list
- Removed HP Diagnostics, HP Network Check, HP App Helper
- Kept ONLY telemetry and advertisement services

**Safety Features:**
- Automatic backup before any changes
- Easy one-click rollback
- Zero hardware service modifications
- OMEN services explicitly protected

### Version Info

- **Version**: 2.0.0
- **Previous Version**: 1.0.0
- **Script Name**: HP Telemetry & Ad Blocker (previously HP Debloater)
- **Target**: HP Laptops with OMEN Gaming Hub
- **Compatibility**: Windows 10/11
